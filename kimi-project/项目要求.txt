请在当前目录创建一个"学生管理系统"项目，要求如下：

### 1. 技术栈
- GUI框架：tkinter（使用ttk现代化组件 + 自定义样式美化）
- 数据库：SQLite3（本地文件存储）
- 数据生成：faker库生成中文真实数据
- 图表：matplotlib集成到tkinter中显示统计图表

### 2. 核心功能模块
**学生信息管理：**
- 增：添加学生（学号、姓名、性别、年龄、班级、专业、入学日期、成绩）
- 删：删除选中记录（带确认对话框）
- 改：双击表格行弹出编辑窗口
- 查：支持按姓名模糊搜索、按班级/专业筛选、按成绩范围筛选

**数据展示：**
- 使用Treeview表格展示，支持：
  - 列宽自适应/手动调整
  - 点击表头排序（升序/降序切换）
  - 行悬停高亮效果
  - 分页显示（每页20条，带翻页按钮）

**统计可视化：**
- 班级人数分布饼图
- 成绩分布直方图
- 各专业平均成绩对比柱状图
- 图表支持鼠标悬停显示具体数值

### 3. 界面美观要求（重点）
- **配色方案**：现代深色主题（参考VS Code暗色主题：#1e1e1e背景，#252526面板，#007acc强调色）
- **字体**：微软雅黑，标题14px加粗，内容12px
- **布局**：
  - 顶部：工具栏（图标按钮：添加、删除、搜索、刷新、生成数据）
  - 左侧：筛选面板（班级下拉框、专业下拉框、成绩区间滑块）
  - 中央：数据表格（带交替行背景色）
  - 右侧：统计图表区（可切换图表类型）
  - 底部：状态栏（显示总记录数、当前选中记录、操作提示）

**交互细节：**
- 所有按钮带悬停效果（颜色变化+鼠标手型）
- 操作成功/失败右下角弹出toast提示（自动消失）
- 添加/编辑使用模态对话框（带表单验证：学号唯一性检查、年龄范围1-100）
- 加载数据时显示进度条动画

### 4. 数据生成要求
- 自动生成5个班级（计算机一班/二班、软件工程一班/二班、人工智能班）
- 生成200条随机学生记录：
  - 姓名：随机中文姓名（faker）
  - 学号：2024 + 班级代码 + 三位流水号（如202401001）
  - 成绩：正态分布随机数（均值75，标准差15，范围0-100）
  - 入学日期：2021-2024年间随机日期

### 5. 项目结构
student_management_system/
├── main.py              # 程序入口，主窗口
├── database.py          # 数据库连接、CRUD操作类
├── data_generator.py    # 伪造数据生成器
├── ui_components.py     # 自定义UI组件（按钮样式、对话框、图表框架）
├── utils.py             # 工具函数（验证、格式化）
├── requirements.txt     # 依赖：faker, matplotlib, numpy
└── students.db          # SQLite数据库（运行时创建）

### 6. 代码规范
- 使用面向对象编程，MainApp类继承tk.Tk
- 数据库操作使用参数化查询防SQL注入
- 所有类和方法写中文docstring
- 异常处理：数据库连接失败、文件占用等错误友好提示

### 7. 启动效果
运行main.py后：
1. 自动初始化数据库和表结构
2. 检测如果数据表为空，自动调用生成器插入200条测试数据（带进度条）
3. 主窗口居中显示，默认最大化窗口但保留最小尺寸限制（800x600）
4. 初始状态显示全部数据，右侧默认显示班级分布饼图

请确保代码可以直接运行，不要遗留未实现的占位函数。
